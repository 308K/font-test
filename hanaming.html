This is a page which can show all the characters in Unicode.

How to use:
Please copy or print in the characters you want the software to show to you into the textarea.

Warnings:
If you see an "Unencoded", a smiling face with leftward eyes, a seedling, or a rabbit, it is shown that the character is not valid.
From Plane4 to Plane15, the characters are reserved now, so please do not use this software to show the invalid ones.
